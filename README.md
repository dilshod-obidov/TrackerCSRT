# OpenCV Trackers


## Setup
```bash
python3 -m venv myenv
source myenv/bin/activate
pip install -r requirements.txt
```

## Run
```bash
python3 main.py --source videos/video.mp4 --tracker CSRT
                                             BOOSTING 
                                             MIL
```
